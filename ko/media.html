Violas 플랫폼 코어 로드맵 #2 전체 버전

2019년 1월 15일 Violas 플랫폼 개발 팀

    비전 협회 파트너들 블록체인 탐색기 미디어 개발자 백서
    日本語 EN

Violas 로드맵 #2에서 우리는 아래 세 가지에 초점을 맞춘다: 1. 우리 글로벌 전략 우선 순위에 맞춘 메인넷 (mainnet) 특성을 설명; 2) Violas 커뮤니티 제공 피드백을 기반으로, Violas 테스트넷에서 프리-메인넷 (pre-mainnet) 으로 그리고 결국 메인넷으로 원활하게 넘어가는 데 필요한 출시 (launch) 표준들을 정의; 3) Violas 플랫폼 위에 자기 Libra 호환 에코시스템을 손쉽게 구축하려는 사람들의 접근성과 개방성에 초점을 맞추어 우리 플랫폼 강점을 Violas 커뮤니티에게 교육. Violas 블록체인은 Libra 블록체인의 ‘맞춤 배치’ (custom deployment) 다.

Violas 로드맵 #1 돌아보기

Violas 로드맵 #1은 Libra 테스트넷 맞춤 배치 위에 구축된 Violas 플랫폼의 기본 기능들을 포함하기 위해 만들어졌다. 목표 달성은 어려운 일이었지만, Violas 로드맵 #1에 나와있던 목표들을 일정대로 달성할 수 있어서 기뻤다. 달성한 목표들은 다음과 같다:

  • 1. Violas 플랫폼 구축 기반이 된 Libra 테스트넷의 맞춤 배치
  • 2. Violas 플랫폼 구축 기반이 된 Libra 테스트넷 코드 변경
  • 3. Bitcoin_core 버전 0.19.1 출시
  • 4. Libra 호환 지갑을 처음부터 개발 (Calibra Wallet은 아직 오픈 소스가 아님)
  • 5. Libra 테스트넷이 지원하지 않는 Libra 호환 스테이블코인 발행
  • 6. Bitcoin 블록체인 상 ‘프루프 오브 프루프’ 프로토콜 개발
  • 7. Bitcoin과 Libra 간 크로스 체인 기능들 통합
  • 8. Violas 관리 (governing) 정책을 Violas 지갑에 통합
  • 9. 기존 웹, 모바일 앱들과 손쉽게 통합될 수 있는 SDK 개발
  • 10. Libra 호환 스테이블코인 발행에 적절한 규제 요건과 ‘스마트 스테이블코인 발행’ (SSO) 지갑 구축
  • 11. Ethereum 에 개발된 DeFi 프로토콜 (i.e. DEX)를 채택해서 이를 MOVE에 적용
  • 12. Libra 호환 스테이블코인 포함 Bitcoin, Libra, Violas 용 브라우저 구축

Violas 플랫폼 구축은 어려움이 많다. 모든 야심적인 프로젝트에 보통 있는 기술적인 어려움에 더해, 우리 경우는 Libra 테스트넷과 Bitcoin 메인넷의 예상 불능 개발 스케줄에 의존한다는 어려움이 있었다. 특히 어려운 점은 Libra 블록체인 기술 자체였다. Libra 블록체인이 쓰고 있는 어떤 기술들은 아직 알파 버전 상태여서 제3자 소프트웨어 구성 요소들이 지원하고 있지 않다. 예를 들면, 우리 Windows 지갑을 지원하는 Libra 클라이언트를 Windows에 만들려 했지만, 이 제한 때문에 불가능 했다. Libra가 쓰는 해시 알고리즘 SHA3-256은OpenSSL 1.1.1 버전 (TLSv1.3)을 필요로 하는데 Python 3.7.x 지원은 아직 초기 단계에 있다.

Violas 메인넷 기능:

  • 지갑 세 개를 통해 Violas 테스트넷에서 라이브 상태인 Violas 메인넷에 필요한 모든 주요 기능들을 성공적으로 개발 했다.

우리는 Violas 플랫폼과 기존 제3자 공급자들 간 통합 또는 사용이 쉬운지에 대한 피드백을 Violas 커뮤니티에 적극적으로 요청하고 있다. 아래 주제들에 대한 제안을 violas.team@violas.io 으로 보내주기 부탁 한다.

  • 메세징 앱 공급자로부터 권한 부여 없이 메세징 앱과 창조적 통합
  • 공유 경제 앱들과 간단한 SDK 또는 API 통합
  • 돈 이체 앱 또는 서비스와 간단한 SDK 또는 API 통합
  • 결제 앱 또는 서비스와 간단한 SDK 또는 API 통합
  • 현 Libra 호환 스테이블코인과 간단한 SDK 또는 API 직접 통합
  • 중앙 집중이든 분산이든 www.coinbase.com 같은 거래소를 통해 현 Libra 호환 스테이블코인과 간단한 SDK 또는 API 통

우리는 또한 Violas 템플레이트를 기존 제3자 공급자들과 통합하기 위해 글로벌 커뮤니티 화이트 라벨 파트너들을 적극적으로 초청하고 있다. 다음 주제들에 관심 있는 파트너는violas.team@violas.io 로 연락 부탁 한다.

  • Libra 호환 스테이블코인을 지역 커뮤니티에 추가할 지역 메세징 앱 공급자
  • Libra 호환 스테이블코인 결제 옵션을 추가할 자동차 타기 공유, 식품 배달, 아파트/주택 렌탈, 자전거 공유, 사무실 공유 등 공유 경제 앱
  • 한 통화를 다른 통화로 교환하는 데 Libra 호환 스테이블코인 거래/이체 옵션을 사용할 국제 돈 이체 앱 또는 서비스
  • 신뢰할 수 있는 제3자 없이 ‘피어 투 피어‘ 직접 결제를 하는 결제 앱 또는 서비스
  • 개발 중인 나라/지역 특히 작은 나라/지역 ‘중앙 은행 디지털 통화‘ (CBDC)
  • Libra 호환 스테이블코인 거래

사용된 용어 정의: Libra 호환 스테이블코인은 Libra와 비슷한 표준으로 발행된 디지털 통화다. 즉, 스테이블코인 발행자는 스테이블코인을 1:1로 뒷받침하는 예비금을 동일 액수 피아트 통화로 보유해야 한다. 또한, 발행자는 지역 규제 당국 또는 기관에서 요구하는 적법 라이선스도 가지고 있어야 한다.

  • 설명 1: 메세징 앱 예: Messenger, Instagram, WhatsApp, Line, WeChat, Messages, Viber, Snapchat, Telegram, Google Hangouts 등
  • 설명 2: 공유 경제 앱 예: Uber, Lyft, DiDi, Airbnb, Homestay, HomeAway, FlipKey, Wimdu, Doordash, Grab 등
  • 설명 3: 돈 이체 앱 또는 서비스 예: IBM World Wire, WorldRemit, OFX, XE, TransferWise, Western Union, CurrencyTransfer, InstaReM, MoneyGram, Ria, Walmart 등
  • 설명 4: 결제 앱 예: Square Cash, Zelle, Google Pay, Facebook Pay, Apple Pay, PayPal, Venmo, Samsung Pay, Circle Pay, Xoom, Facepay at https://facepay.io/ 등
