Libra Core概述

Libra Core 是Libra协会发布的基于Libra协议的开源实现的官方项目名称。

  • 该软件是Libra协议和Move语言的第一个实现。
  • Libra Core包括验证器和客户端功能。
  • Libra Core是用Rust编写的。
